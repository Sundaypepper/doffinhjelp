# doffinhjelp.no

Landingsside for doffinhjelp.no — markedsf&oslash;rer Anbudsradar (anbudsradar.no).

## Deploy

Statisk site. Deploy til Netlify:

1. Opprett nytt repo p&aring; GitHub
2. Push filene
3. Koble til Netlify og pek domenet `doffinhjelp.no` dit

## Filer

- `index.html` — Landingssiden (SEO-optimalisert, all CSS inline)
- `netlify.toml` — Netlify-konfigurasjon med security headers
- `robots.txt` — Tillat alle crawlere
- `sitemap.xml` — Sitemap for Google
